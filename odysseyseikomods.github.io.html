The Story

Where Space Meets Craft

Every watch is hand-assembled — sourcing quality movements, custom dials, cases, and bracelets, then meticulously putting them together to create something no catalogue can offer. Each build is a unique journey.

How It's Made

The Process

01
Source

Quality components are sourced from trusted suppliers — movements, dials, cases, and bracelets, all selected for their precision and finish.

02
Design

Each build is designed as a cohesive aesthetic — colour, texture, and proportion are all considered before a single component is touched.

03
Assemble

Careful hand-assembly using proper tools. Movement service, dial fitting, stem and crown installation — all done with patience and precision.

04
Quality Check

Every finished watch is inspected — timekeeping accuracy, crown function, case back seal. Only when it meets standard does it leave the bench.
